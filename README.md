# HouseRizz Seller iOS App

This the Seller app for [HouseRizz](https://houserizz-com.vercel.app) a tool that lets you Redesign your home with  AI & AR.

## Features

- [x]  Email/Password User Firebase Authentication
- [x]  Google User Firebase Authentication
- [x]  Apple User Firebase Authentication
- [x]  Add Item to CloudKit
- [x]  Upload USDZ Model
- [x]  Fetch Item
- [x]  Delete Item
- [x]  Item Detail View
- [ ]  Order Management
